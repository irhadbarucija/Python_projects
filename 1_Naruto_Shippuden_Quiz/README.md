Naruto Shippuden Quiz 1.0

A clean and interactive console quiz game about the Naruto Shippuden universe, developed in Python.  
Players will answer trivia questions and receive a rank based on their knowledge.

Project Information:

**Developer:** `irhadbarucija`
**Project Type:** Console-based quiz
**Language:** Python

Features:

- 10 trivia questions  
- Score-based ranking (Genin, Chunin, Jonin)  
- Exit quiz anytime (`q`)  
- Answer explanations for correct answers  
- Case-insensitive input  
- Clean question structure using Python lists & loops
